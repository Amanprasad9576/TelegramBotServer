## setup instruction 

1 . clone the repository

2. Install the dependence 

   '''bash 

  npm install 

  '''
3. Create a new file ' .env' in the root directory of the project 
and add the folloeing

 '''bash 

 BOT_TOKEN = your_bot_token

 '''
 4. start the server 

 '''bash 

 node index.js 

 '''
